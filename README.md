# Comnandos Básicos para usar no Git.
## Descrição dos comandos.
### git clone: lona um repositório remoto.
### git add: adiciona arquivos ao stage (preparação).
### git commit: usando para criar uma mensagem para descrever o código que está sendo upado.
### git push: envia os commits para o repositório remoto.
### git branch: lista as branches.
### git branch (nome): cria uma branch nova.
### git checkout (nome) ou git switch (nome): ambos troca de branch.
### git push -u origin (nome): envia a nova branch para o repositório remoto.
### git commit --amend -m "texto": altera a última mensagem do commit.
### git add + git commit --amend --no-edit: adiciona arquivos esquecidos ao último commit.
### git reset HEAD~1: desfaz o último commit
### git reset --hard HEAD~1: desfaz o último commit apagando as alterações feitas.
### git revert (commit-sha): reverte o último commit enviado (push) com segurança.
### git status: vereficar todo o status do git.
### git log --oneline: ver todo o histórico.
